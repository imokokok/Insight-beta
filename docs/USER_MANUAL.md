# 用户手册

## 快速开始

1. **连接钱包** - 点击右上角 "Connect Wallet" 按钮
2. **浏览数据** - 在 Dashboard 查看聚合价格数据
3. **查看协议** - 访问不同协议的详细信息
4. **完成新手引导** - 首次访问时会自动显示新手引导

## 核心功能

- **多协议价格监控** - 实时聚合 9+ 协议的价格数据
- **价格对比** - 跨协议价格比较和偏差分析
- **UMA 断言** - 创建、争议和结算断言
- **告警通知** - 价格偏差告警和通知
- **安全监控** - 价格操纵检测和安全告警
- **SLO 监控** - 服务级别目标监控

## 常见问题

**如何查看价格对比？**

- 进入 Dashboard 或 Comparison 页面
- 选择交易对（如 ETH/USD）
- 查看不同协议的价格对比

**什么是价格偏差？**

- 不同协议间的价格差异
- 高偏差可能表示市场波动或预言机问题
- 系统会自动检测并告警

**如何设置告警？**

- 进入 Alerts 页面
- 创建新的告警规则
- 设置价格偏差阈值
- 选择通知渠道（Slack、Telegram、邮件）

**如何重新查看新手引导？**

- 在设置页面找到 "重新查看引导" 按钮
- 或使用 `OnboardingReset` 组件

**支持哪些语言？**

- 中文、英语、西班牙语、法语、韩语
- 在页面右上角切换语言
