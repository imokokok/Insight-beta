# 用户手册

## 快速开始

1. **连接钱包** - 点击右上角 "Connect Wallet" 按钮
2. **浏览数据** - 在 Dashboard 查看聚合价格数据
3. **查看协议** - 访问不同协议的详细信息

## 核心功能

- **多协议价格监控** - 实时聚合 8+ 协议的价格数据
- **价格对比** - 跨协议价格比较和偏差分析
- **UMA 断言** - 创建、争议和结算断言
- **告警通知** - 价格偏差告警

## 页面导航

| 页面      | 功能               |
| --------- | ------------------ |
| Dashboard | 系统概览、价格聚合 |
| Protocols | 各协议详情         |
| UMA       | UMA 断言和争议     |
| Alerts    | 告警配置           |
| Audit     | 操作日志           |

## 支持协议

UMA、Chainlink、Pyth、Band、API3、RedStone、Switchboard、Flux、DIA

## 常见问题

**如何查看价格对比？**

- 进入 Dashboard 或 Comparison 页面
- 选择交易对（如 ETH/USD）

**什么是价格偏差？**

- 不同协议间的价格差异
- 高偏差可能表示市场波动或预言机问题
