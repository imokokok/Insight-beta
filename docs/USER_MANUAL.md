# 用户手册

## 1. 概述

Insight 是一个先进的预言机监控和争议解决界面，允许用户实时可视化预言机数据，参与争议解决，并通过现代化的 Web 界面与协议进行无缝交互。

## 2. 核心功能

### 2.1 实时监控

- 可视化预言机数据趋势、交易量和同步状态
- 交互式图表展示每日断言数、累计安全价值等关键指标
- 监控索引器健康状态和同步进度

### 2.2 争议解决

- 浏览当前活跃争议列表
- 参与投票和提议结算
- 查看争议详情、证据和时间线

### 2.3 断言管理

- 创建新断言
- 追踪断言状态和进展
- 管理个人断言和争议

### 2.4 钱包集成

- 支持通过 MetaMask、Rabby 等 Web3 钱包连接
- 支持多链部署（Polygon、Arbitrum、Optimism、Local testnets）
- 安全的交易签名和执行

## 3. 快速开始

### 3.1 连接钱包

1. 打开应用程序
2. 点击右上角的「连接钱包」按钮
3. 选择您的钱包提供商（如 MetaMask）
4. 按照钱包提示完成连接

### 3.2 浏览断言

1. 在首页，您可以看到最近的断言列表
2. 使用筛选器按状态、链或关键字搜索断言
3. 点击断言卡片查看详情

### 3.3 创建断言

1. 点击「新建断言」按钮
2. 填写断言详情：
   - 协议名称
   - 市场问题
   - 断言内容
   - 保证金金额
   - 挑战期时长
3. 点击「创建断言」按钮
4. 按照钱包提示签署交易

### 3.4 争议断言

1. 打开断言详情页面
2. 如果断言处于「待处理」状态且在挑战期内，点击「争议此断言」按钮
3. 填写争议理由
4. 点击「确认争议」按钮
5. 按照钱包提示签署交易

### 3.5 结算断言

1. 打开断言详情页面
2. 如果断言已过挑战期，点击「结算断言」按钮
3. 选择结算结果（有效/真实或无效/虚假）
4. 点击「确认结算」按钮
5. 按照钱包提示签署交易

### 4.4 审计

- 查看审计日志

### 4.5 我的断言

- 查看和管理您创建的所有断言

### 4.6 我的争议

- 查看和管理您发起的所有争议

- 查看您关注的断言

## 5. 高级功能

### 5.1 关注列表

1. 在断言卡片上点击星形图标
2. 断言将被添加到您的关注列表
3. 您可以在「关注列表」页面查看所有关注的断言

### 5.2 收益计算器

1. 导航到「监控台 > 工具」页面
2. 选择您要计算的场景（断言或争议）
3. 输入保证金金额
4. 查看潜在收益和投资回报率

### 5.3 多链切换

1. 确保您的钱包已连接
2. 导航到断言或争议详情页面
3. 如果需要，系统会提示您切换到相应的链
4. 按照钱包提示完成链切换

## 6. 常见问题

### 6.1 如何连接钱包？

1. 点击右上角的「连接钱包」按钮
2. 选择您的钱包提供商
3. 按照钱包提示完成连接

### 6.2 为什么我无法创建断言？

- 确保您的钱包已连接
- 确保您有足够的资金支付交易费用和保证金
- 确保您在正确的链上

### 6.3 如何参与争议？

- 确保断言处于「待处理」状态
- 确保您在挑战期内
- 确保您有足够的资金支付争议保证金

### 6.4 如何结算断言？

- 确保断言已过挑战期
- 确保您有足够的资金支付交易费用
- 按照系统提示选择结算结果

### 6.5 如何查看交易详情？

- 在断言或争议详情页面，点击交易哈希链接
- 您将被重定向到相应链的区块链浏览器
