## 账户与访问

### 是否需要创建账户才能使用 Insight？

不需要。Insight 的大部分核心功能（如查看预言机数据、价格比较、数据分析等）无需登录即可使用。但某些高级功能（如保存告警规则、个性化设置）可能需要连接钱包。

### 支持哪些钱包？

Insight 支持以下钱包：

- MetaMask
- Phantom
- WalletConnect（通过 WalletConnect 可以连接更多钱包）

### 如何切换语言？

点击页面顶部的语言切换按钮，选择「中文」或「English」即可。Insight 会记住你的语言偏好。

### 平台支持哪些浏览器？

推荐使用最新版本的 Chrome、Firefox、Safari 或 Edge 浏览器。

## 功能使用

### 如何查看预言机数据？

1. 点击左侧导航栏的「预言机」→「仪表板」
2. 你会看到所有支持的预言机协议概览
3. 点击任意价格源查看详细数据和历史走势图

### 如何比较多个预言机协议的价格？

1. 点击「预言机」→「对比」
2. 选择你想要比较的价格源和协议
3. 查看实时价格对比、偏差热力图和延迟分析

### 跨链分析可以做什么？

跨链分析功能可以帮助你：

- 比较不同区块链上同一资产的价格
- 发现跨链套利机会
- 分析不同链间的价格相关性
- 查看流动性分布

### 数据探索功能有什么用？

数据探索帮助你：

- 搜索和发现新的价格源
- 查看市场概览和热门价格源
- 探索特定协议或地址的详细信息
- 查找历史数据

## 数据相关

### Insight 支持哪些预言机协议？

目前支持以下预言机协议：

- **Chainlink** - 行业标准价格源
- **Pyth** - 低延迟机构数据
- **RedStone** - 专为 L2 优化的模块化预言机
- **UMA** - 乐观预言机，支持断言和争议

### 数据更新频率是多少？

数据更新频率取决于具体的预言机协议：

- 实时价格：通过 WebSocket 实时推送
- 历史数据：按小时/天聚合
- 分析数据：定期更新

### 价格偏差是如何计算的？

价格偏差是多个预言机协议报价之间的相对差异，通常以百分比表示。我们计算：

- 最大值与最小值的差异
- 相对于平均值的偏差
- 标准差等统计指标

### 可以导出数据吗？

可以。在偏差分析等页面，你可以将数据导出为 CSV 或 Excel 格式。

## 告警相关

### 如何创建告警？

1. 点击左侧导航栏的「告警」
2. 点击「创建告警」按钮
3. 配置告警规则（价格源、触发条件、通知方式）
4. 保存并启用告警

### 支持哪些告警触发条件？

支持多种触发条件：

- 价格偏差超过指定百分比
- 价格高于或低于某个阈值
- 数据更新延迟超过指定时间
- 多个条件的组合

### 可以通过哪些方式接收告警通知？

支持多种通知渠道：

- 邮件
- Telegram 机器人
- Slack Webhook
- 自定义 HTTP Webhook

你可以同时配置多个通知渠道。

### 如何查看告警历史？

在告警页面，点击「历史记录」标签即可查看所有触发过的告警。

## 技术问题

### Insight 使用什么技术栈？

- **前端框架**: Next.js 16 + React 19
- **编程语言**: TypeScript 5.7
- **样式**: Tailwind CSS 3.4
- **数据库**: PostgreSQL + Supabase
- **区块链**: viem

### API 访问如何认证？

API 使用 API Key 认证。在请求头中包含你的 API Key：

```
Authorization: Bearer YOUR_API_KEY
```
