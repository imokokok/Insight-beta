## 首次访问

### 访问平台

在浏览器中打开 Insight 平台地址（例如：`https://insight.yourdomain.com`）。访问后会自动重定向到「仪表板」页面。

### 连接钱包（可选）

虽然 Insight 平台的大部分功能无需钱包连接即可使用，但连接钱包可以解锁更多高级功能：

1. 点击页面底部的「连接钱包」按钮
2. 选择你的钱包（MetaMask、Phantom 或 WalletConnect）
3. 按照钱包提示完成连接

### 切换语言

Insight 支持中英文双语：

1. 点击语言切换器
2. 选择「中文」或「English」

## 仪表板概览

访问平台后，你将看到仪表板页面，这是你的数据中心。仪表板包含三个主要标签页：

### 仪表板标签页

页面顶部显示关键指标：

- **活跃价格源** - 当前监控的预言机价格源数量
- **平均偏差** - 多协议间的平均价格偏差
- **告警数量** - 当前活跃的告警规则数
- **网络健康度** - 整体预言机网络健康评分

中央区域展示实时价格走势图和偏差热力图，帮助你快速了解市场状况。

### 预言机对比标签页

在此标签页中，你可以：

1. 选择多个预言机协议进行价格对比
2. 查看实时价格并排展示
3. 分析价格热力图和延迟数据
4. 切换不同时间范围查看历史对比

### 偏差分析标签页

在此标签页中，你可以：

1. 深入分析各协议间的价格偏差
2. 查看偏差趋势图和热力图
3. 识别异常偏差事件
4. 导出分析数据

## 预言机监控

左侧导航栏的「预言机监控」分组提供完整的预言机数据分析功能。

### 跨链分析

1. 点击左侧导航栏的「跨链」
2. 选择要比较的区块链网络
3. 查看：
   - 跨链价格对比
   - 相关性矩阵
   - 价格一致性监控
   - 流动性分析（模拟数据）

> **注意**: 跨链分析功能仅用于监控预言机数据质量，不提供交易建议。价格差异可能由数据延迟、流动性差异等因素造成。

### 可靠性监控

1. 点击左侧导航栏的「可靠性」
2. 查看各预言机协议的可靠性评分
3. 监控实时状态和历史记录
4. 查看详细的可靠性分析报告

### 数据探索

1. 点击左侧导航栏的「探索」
2. 使用搜索功能查找特定价格源或协议
3. 查看：
   - 市场概览
   - 热门价格源
   - 协议活动
   - 最近异常

## 协议分析

左侧导航栏的「协议分析」分组提供对各个预言机协议的深入分析。

### Chainlink 分析

1. 点击左侧导航栏的「Chainlink」
2. 查看 Chainlink 喂价数据
3. 监控节点状态和 Gas 费用
4. 分析 OCR 协议性能

### Pyth 分析

1. 点击左侧导航栏的「Pyth」
2. 查看 Pyth 低延迟价格数据
3. 监控 Publisher 状态和信任评分
4. 分析价格置信区间

### API3 分析

1. 点击左侧导航栏的「API3」
2. 查看 dAPIs 价格数据
3. 监控 Airnode 节点状态
4. 查看 OEV 事件

### Band Protocol 分析

1. 点击左侧导航栏的「Band」
2. 查看 Band Protocol 数据
3. 监控跨链桥状态
4. 验证数据源可靠性

### UMA 争议分析

1. 点击左侧导航栏的「UMA」（争议分析）
2. 查看争议历史记录
3. 分析争议趋势和投票者行为
4. 监控协议 TVL 变化

## 设置告警

左侧导航栏的「工具」分组包含告警中心功能。

### 访问告警中心

1. 点击左侧导航栏的「告警中心」
2. 你会看到多个标签页：全部、价格异常、跨链、安全、规则、通道、分析

### 创建告警规则

1. 在告警中心，点击「规则」标签
2. 点击「创建告警」按钮
3. 配置告警规则：

**基本信息：**

- 告警名称
- 选择价格源（例如 BTC/USD）
- 选择监控的协议

**触发条件：**

- 价格偏差超过 X%
- 价格高于/低于 Y
- 数据陈旧超过 Z 分钟

**通知方式：**

- 邮件
- Telegram
- Slack Webhook
- 自定义 Webhook

### 管理告警

在告警中心，你可以：

- 查看所有告警按类型分组
- 启用/禁用告警规则
- 编辑告警配置
- 查看告警历史和趋势分析
- 测试通知渠道
- 批量处理告警
