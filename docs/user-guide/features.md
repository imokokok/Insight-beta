## 预言机监控

### 功能概述

预言机监控是 Insight 平台的核心功能组，提供完整的预言机数据分析和监控能力。

### 子功能

#### 仪表板

仪表板是 Insight 平台的核心入口，提供所有预言机数据的统一视图。

**主要功能：**

- **关键指标概览** - 实时显示活跃价格源数量、平均偏差、告警数量、网络健康度
- **快速操作面板** - 一键访问热门功能
- **实时价格图表** - 展示主要加密货币价格走势
- **偏差热力图** - 直观显示各协议间价格偏差
- **最近告警** - 查看最新触发的告警事件

**使用场景：**

- 日常监控预言机网络整体状态
- 快速发现异常情况
- 一键跳转到详细分析页面

#### 预言机对比

实时比较多个预言机协议的价格数据。

**主要功能：**

- **实时价格对比** - 并排展示多协议价格
- **价格热力图** - 可视化价格差异
- **延迟分析** - 比较各协议数据更新延迟
- **历史对比** - 查看历史价格对比
- **价格源选择** - 灵活选择要比较的价格源和协议

**使用场景：**

- 选择最优价格源
- 验证价格准确性
- 分析协议性能差异

#### 偏差分析

深入分析各预言机协议的数据质量、偏差和性能。

**主要功能：**

- **实时偏差监控** - 监控多协议间价格偏差
- **偏差趋势图** - 查看历史偏差变化
- **偏差热力图** - 可视化价格源间偏差分布
- **异常事件检测** - 自动识别异常偏差
- **统计摘要** - 偏差均值、最大值、标准差等统计数据
- **数据导出** - 支持 CSV/Excel 导出分析结果

**使用场景：**

- 评估预言机数据可靠性
- 发现系统性偏差问题

#### 跨链

分析多条区块链上的预言机数据，发现跨链机会。

**主要功能：**

- **跨链价格比较** - 比较不同链上同一资产的价格
- **价格差异监控** - 监控跨链价格差异
- **历史趋势** - 查看跨链价格差异历史
- **相关性分析** - 展示不同链/价格源间的相关性
- **价格一致性监控** - 监控各协议/链的价格一致性
- **流动性分析** - 各协议/链的流动性分布（模拟数据）

> **免责声明**: 本功能仅用于监控预言机数据质量，不提供交易建议。价格差异可能由数据延迟、流动性差异等因素造成，不构成套利机会。
>
> **注意**: 流动性分析功能当前使用模拟数据（mock data），非实时流动性数据。

**使用场景：**

- 评估跨链价格一致性
- 分析多链市场结构
- 评估预言机可靠性
- 监控数据质量

#### 可靠性

监控和评估预言机网络的可靠性指标。

**主要功能：**

- **可靠性评分** - 综合评估各预言机协议的可靠性
- **实时状态监控** - 监控预言机节点在线状态
- **历史可靠性记录** - 查看可靠性历史趋势
- **可靠性分析报告** - 详细的可靠性分析数据

**使用场景：**

- 选择可靠的预言机协议
- 监控预言机服务质量
- 风险管理和评估

#### 探索

探索和搜索预言机数据，发现市场趋势。

**主要功能：**

- **市场概览** - 预言机市场整体健康度和关键指标汇总
- **热门价格源** - 最受关注的价格源排行和趋势分析
- **全局搜索** - 搜索价格源、协议、地址，支持智能建议
- **协议探索** - 所有支持的预言机协议列表和详情
- **地址探索** - 按合约地址查询和相关数据活动

**使用场景：**

- 发现新的价格源
- 研究特定协议
- 查找历史数据

## 协议分析

### 功能概述

协议分析提供对各个预言机协议的深入分析和监控。

### 子功能

#### Chainlink 分析

深入分析 Chainlink 预言机网络的数据。

**主要功能：**

- **喂价监控** - 监控 Chainlink 价格喂价数据
- **节点状态** - 查看 Chainlink 节点运行状态
- **Gas 分析** - 分析 Chainlink 交易 Gas 费用
- **心跳监控** - 监控喂价更新频率
- **OCR 协议分析** - 分析 OCR 协议性能

**使用场景：**

- 监控 Chainlink 喂价质量
- 评估 Chainlink 网络性能

#### Pyth 分析

分析 Pyth 预言机网络的低延迟数据。

**主要功能：**

- **Publisher 监控** - 监控 Pyth Publisher 的状态和信任评分
- **置信区间分析** - 分析价格置信区间
- **价格更新统计** - 价格更新频率和延迟统计
- **Hermes 监控** - 监控 Hermes 中继节点状态

**使用场景：**

- 使用 Pyth 低延迟数据
- 评估 Publisher 可靠性

#### API3 分析

分析 API3 去中心化 API 服务。

**主要功能：**

- **Airnode 监控** - 监控 Airnode 节点状态和离线事件
- **dAPIs 价格** - 查看 dAPIs 价格数据和历史
- **OEV 监控** - 价格更新事件监控
- **跨链比较** - API3 与其他协议的价格比较

**使用场景：**

- 使用 API3 dAPIs 数据
- 监控 Airnode 运行状态

#### Band Protocol 分析

分析 Band Protocol 跨链数据预言机。

**主要功能：**

- **数据源验证** - 验证数据源的可靠性
- **跨链桥监控** - 监控 Band 跨链桥状态
- **聚合验证** - 验证价格聚合的正确性
- **IBC 传输监控** - 监控 IBC 数据传输

**使用场景：**

- 使用 Band Protocol 数据
- 监控跨链数据传输

#### UMA 争议分析

分析 UMA 乐观预言机的争议机制。

**主要功能：**

- **争议历史** - 查看 UMA 等协议的争议记录
- **争议趋势** - 争议数量和结果趋势图
- **保证金分布** - 争议保证金金额分布
- **争议详情** - 查看具体争议的详细信息
- **投票者分析** - 分析投票者行为
- **TVL 监控** - 监控协议 TVL 变化

**使用场景：**

- 分析争议事件模式
- 评估 UMA 协议安全性

## 工具

### 功能概述

提供实用工具帮助用户更好地管理和使用平台。

### 子功能

#### 告警中心

灵活强大的告警系统，及时通知重要事件。

**主要功能：**

- **告警规则** - 灵活配置多种触发条件（价格偏差、价格阈值、数据陈旧等）
- **通知渠道** - 支持邮件、Telegram、Slack Webhook、自定义 HTTP Webhook
- **告警管理** - 查看、启用/禁用、编辑告警规则
- **告警历史** - 所有触发的告警记录和趋势分析
- **响应时间统计** - 告警响应时间监控和性能分析
- **批量操作** - 支持批量选择和处理告警
- **多维度筛选** - 按严重程度、状态、分组等筛选

**告警类型：**

- **价格异常** - 价格偏差和阈值告警
- **跨链** - 跨链数据异常告警
- **安全** - 安全相关告警

**使用场景：**

- 实时监控价格异常
- 及时发现数据问题
- 自动化运维通知

#### 收藏夹

快速访问常用的价格源和页面。

**主要功能：**

- **快速收藏** - 一键收藏价格源和页面
- **收藏管理** - 查看和管理收藏内容
- **快速访问** - 侧边栏快速访问收藏

**使用场景：**

- 快速访问常用内容
- 个性化工作空间

## 多语言支持

### 功能概述

支持中英文双语，满足不同用户需求。

### 主要功能

- **语言切换** - 一键切换中英文
- **界面本地化** - 完整的界面翻译

### 支持的语言

- 中文 (简体)
- English

## API 访问

### 功能概述

完整的 RESTful API，支持程序化访问。

### 主要功能

- **REST API** - 标准 RESTful 接口
- **实时数据** - WebSocket/SSE 实时推送
- **Swagger 文档** - 交互式 API 文档
- **认证支持** - API Key 认证

### API 端点

- `/api/oracle/*` - 预言机数据相关
- `/api/comparison/*` - 价格比较相关
- `/api/cross-chain/*` - 跨链分析相关
- `/api/alerts/*` - 告警系统相关
- `/api/explore/*` - 数据探索相关

### 使用场景

- 集成到自己的系统
- 自动化数据分析
- 构建自定义应用
