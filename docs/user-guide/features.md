## 仪表板

### 功能概述

仪表板是 Insight 平台的核心入口，提供所有预言机数据的统一视图。

### 主要功能

- **关键指标概览** - 实时显示活跃价格源数量、平均偏差、告警数量、网络健康度
- **快速操作面板** - 一键访问热门功能
- **实时价格图表** - 展示主要加密货币价格走势
- **偏差热力图** - 直观显示各协议间价格偏差
- **最近告警** - 查看最新触发的告警事件

### 使用场景

- 日常监控预言机网络整体状态
- 快速发现异常情况
- 一键跳转到详细分析页面

## 预言机分析

### 功能概述

深入分析各预言机协议的数据质量、偏差和性能。

### 子功能

#### 偏差分析

- **实时偏差监控** - 监控多协议间价格偏差
- **偏差趋势图** - 查看历史偏差变化
- **偏差热力图** - 可视化价格源间偏差分布
- **异常事件检测** - 自动识别异常偏差
- **统计摘要** - 偏差均值、最大值、标准差等统计数据
- **数据导出** - 支持 CSV/Excel 导出分析结果

#### 争议分析

- **争议历史** - 查看 UMA 等协议的争议记录
- **争议趋势** - 争议数量和结果趋势图
- **保证金分布** - 争议保证金金额分布
- **争议详情** - 查看具体争议的详细信息

#### API3 分析

- **Airnode 监控** - 监控 Airnode 节点状态和离线事件
- **dAPIs 价格** - 查看 dAPIs 价格数据和历史
- **OEV 监控** - 价格更新事件监控（实验性功能）
- **跨链比较** - API3 与其他协议的价格比较

#### Band Protocol 分析

- **数据源验证** - 验证数据源的可靠性
- **跨链桥监控** - 监控 Band 跨链桥状态
- **聚合验证** - 验证价格聚合的正确性

#### Pyth 分析

- **Publisher 监控** - 监控 Pyth Publisher 的状态和信任评分（实验性）
- **置信区间分析** - 分析价格置信区间
- **价格更新统计** - 价格更新频率和延迟统计

### 使用场景

- 评估预言机数据可靠性
- 发现系统性偏差问题
- 分析争议事件模式

## 价格比较

### 功能概述

实时比较多个预言机协议的价格数据。

### 主要功能

- **实时价格对比** - 并排展示多协议价格
- **价格热力图** - 可视化价格差异
- **延迟分析** - 比较各协议数据更新延迟
- **历史对比** - 查看历史价格对比
- **价格源选择** - 灵活选择要比较的价格源和协议

### 使用场景

- 选择最优价格源
- 验证价格准确性
- 分析协议性能差异

## 跨链分析

### 功能概述

分析多条区块链上的预言机数据，发现跨链机会。

### 子功能

#### 跨链价格比较

- **多链价格对比** - 比较不同链上同一资产的价格
- **价格差异监控** - 监控跨链价格差异
- **历史趋势** - 查看跨链价格差异历史

#### 相关性分析

- **相关性矩阵** - 展示不同链/价格源间的相关性
- **相关性趋势** - 相关性随时间变化
- **热图可视化** - 直观的相关性热图

#### 价格一致性监控

- **价格一致性检查** - 监控各协议/链的价格一致性
- **偏差趋势** - 价格偏差变化趋势
- **异常检测** - 识别价格异常情况

#### 流动性分析（模拟数据）

- **流动性分布** - 各协议/链的流动性分布（模拟数据）
- **流动性趋势** - 流动性变化趋势（模拟数据）
- **深度图** - 订单簿深度可视化（模拟数据）

> **免责声明**: 本功能仅用于监控预言机数据质量，不提供交易建议。价格差异可能由数据延迟、流动性差异等因素造成，不构成套利机会。
>
> **注意**: 流动性分析功能当前使用模拟数据（mock data），非实时流动性数据。

### 使用场景

- 评估跨链价格一致性
- 分析多链市场结构
- 评估预言机可靠性
- 监控数据质量

- **关键指标汇总** - 市场关键统计指标
- **趋势摘要** - 市场趋势快速摘要

#### 热门价格源

- **热门排行** - 最受关注的价格源排行
- **趋势分析** - 热门价格源趋势
- **快速筛选** - 按协议、资产类型筛选

#### 搜索功能

- **全局搜索** - 搜索价格源、协议、地址
- **智能建议** - 搜索建议和自动补全
- **高级筛选** - 多维度筛选条件

#### 协议探索

- **协议列表** - 所有支持的预言机协议
- **协议详情** - 协议详细信息和统计
- **健康状态** - 协议健康度评分

## 告警系统

### 功能概述

灵活强大的告警系统，及时通知重要事件。

### 子功能

#### 告警规则

- **灵活配置** - 多种触发条件
- **价格偏差告警** - 价格偏差超过阈值
- **价格阈值告警** - 价格高于/低于指定值
- **数据陈旧告警** - 数据更新延迟告警
- **多条件组合** - 支持多个条件组合

#### 通知渠道

- **邮件通知** - 发送邮件告警
- **Telegram** - Telegram 机器人通知
- **Slack** - Slack Webhook 通知
- **自定义 Webhook** - 任意 HTTP Webhook
- **多渠道同时** - 支持同时发送到多个渠道

#### 告警管理

- **规则列表** - 查看所有告警规则
- **启用/禁用** - 灵活控制规则状态
- **规则编辑** - 修改已有规则
- **测试通知** - 测试通知渠道是否正常

#### 告警历史

- **历史记录** - 所有触发的告警记录
- **告警详情** - 告警触发时的详细信息
- **趋势分析** - 告警触发趋势
- **导出功能** - 导出告警历史

#### 响应时间统计

- **响应时间监控** - 告警响应时间统计
- **性能分析** - 通知渠道性能
- **SLA 报告** - 服务水平协议报告

### 使用场景

- 实时监控价格异常
- 及时发现数据问题
- 自动化运维通知

## API 访问

### 功能概述

完整的 RESTful API，支持程序化访问。

### 主要功能

- **REST API** - 标准 RESTful 接口
- **实时数据** - WebSocket/SSE 实时推送
- **Swagger 文档** - 交互式 API 文档
- **认证支持** - API Key 认证

### API 端点

- `/api/oracle/*` - 预言机数据相关
- `/api/comparison/*` - 价格比较相关
- `/api/cross-chain/*` - 跨链分析相关
- `/api/alerts/*` - 告警系统相关
- `/api/explore/*` - 数据探索相关

### 使用场景

- 集成到自己的系统
- 自动化数据分析
- 构建自定义应用
