# Contributing to Insight

感谢你的贡献！

## 贡献流程

1. **提交 Issue** - 报告 bug 或建议新功能
2. **Fork 仓库** - 创建你的分支
3. **提交 PR** - 遵循以下规范

## 提交规范

- 使用现在时态 ("Add feature" 而非 "Added feature")
- 使用祈使语气 ("Move cursor to..." 而非 "Moves cursor to...")
- 首行限制在 72 字符以内

## 开发设置

```bash
npm install
npm run dev
npm test
```

详细开发指南请参考 [Development Guide](docs/DEVELOPMENT_GUIDE.md)
