## 功能特性详情

### 1. Airnode 运营商信息
- **运营商名称**: 显示运营机构的名称
- **信誉评分**: 0-100 的评分系统
- **运营统计**: 总节点数、活跃节点数
- **历史表现**: 平均响应时间、趋势分析（改善/稳定/下降）

### 2. 价格更新热力图
- **二维展示**: 按星期（7 天）× 小时（24 小时）展示
- **颜色编码**: 
  - 绿色：正常更新
  - 黄色：警告（延迟较高）
  - 红色：严重延迟
- **交互功能**: 鼠标悬停显示详细统计
- **数据聚合**: 自动聚合相同时段的数据

### 3. dAPI 历史趋势
- **时间范围**: 支持 1 小时、24 小时、7 天、30 天
- **三维度分析**:
  - 价格趋势图（面积图）
  - 更新频率图（折线图）
  - 延迟趋势图（折线图）
- **统计卡片**: 4 个关键指标实时展示
- **趋势分析**: 自动计算价格变化和趋势

### 4. 跨链一致性验证
- **一致性评分**: 0-100 分，4 个等级
  - 优秀（90-100）
  - 良好（75-89）
  - 一般（60-74）
  - 较差（<60）
- **评分算法**:
  - 最大偏差扣分（×5）
  - 平均偏差扣分（×10）
  - 未同步链扣分（×20）
- **双轴图表**: 
  - 左轴：价格（蓝色柱状图）
  - 右轴：偏差（橙色柱状图）
- **同步阈值**: 1% 偏差线标记
- **详细表格**: 展示各链完整数据

## 使用指南

### Airnode 详情页使用
1. 访问 Airnode 列表页
2. 点击任意 Airnode 进入详情页
3. 查看运营商信息和历史表现
4. 分析响应时间趋势图

### 价格更新热力图使用
1. 访问 API3 页面 Analysis 标签
2. 查看热力图了解更新模式
3. 悬停查看特定时段的详细统计
4. 识别更新延迟高发时段

### dAPI 历史趋势使用
1. 在 dAPI 列表页选择特定 dAPI
2. 点击历史趋势入口
3. 切换时间范围查看不同周期
4. 分析价格、更新频率、延迟的关系

### 跨链一致性使用
1. 访问 Analysis 标签页
2. 查看跨链一致性分析卡片
3. 了解各链数据差异
4. 识别数据不同步的链
