# Checklist

## 高优先级检查项

- [x] 协议详情页面能够正常加载真实数据
- [x] `/api/oracle/protocols/[protocol]` API 返回正确的协议信息
- [x] 协议比较页面显示真实性能指标而非 Mock 数据
- [x] `ProtocolCompare.tsx` 不再使用 `generateMockMetrics` 函数

## 中优先级检查项

- [x] 所有 API 响应包含 `isMock` 字段标识数据来源
- [x] 前端在显示 Mock 数据时提示用户
- [x] 可靠性评分 API 在无数据库时返回 Mock 数据而非 503 错误
- [x] 偏差分析 API 在返回 Mock 数据时明确标识

## 数据完整性检查项

- [x] 协议详情包含完整的价格源列表
- [x] 协议详情包含支持链信息
- [x] 协议详情包含性能指标 (延迟、准确性、更新频率)
- [x] 协议比较数据反映真实市场情况

## 用户体验检查项

- [x] 用户能够区分真实数据和 Mock 数据
- [x] Mock 数据使用时有明显的视觉提示
- [x] 数据加载失败时有友好的错误提示
- [x] 页面加载状态正确显示

## API 规范检查项

- [x] 所有 API 返回统一的响应格式
- [x] API 包含正确的错误处理
- [x] API 包含性能日志记录
- [ ] API 文档 (Swagger) 已更新
